       ```python
       def _check_e17(
           output_dir: Path,
           known_models: dict[str, Any],
           registry: "ModelRegistry | None" = None,
       ) -> tuple[list[ValidationIssue], list[ValidationIssue]]:
           """E17: Extension xpath references non-existent base field.

           Returns (errors, warnings) tuple.
           Errors for Tier 1/2 (known/registry models with bad field refs).
           Warnings (W6) for Tier 3 (unknown models -- cannot validate).
           """
       ```

       Implementation:
       - Iterate over all XML files in output_dir.
       - Parse each XML. For each `<record>` element:
         - Check if it has a `<field name="inherit_id">` child with a `ref` attribute. If not, skip (not an inherited view).
         - Extract model name from `<field name="model">` child.
         - Find `<field name="arch">` child. Iterate over all `<xpath>` elements inside.
         - For each xpath, extract field reference using regex: `r"field\[@name=['\"](\w+)['\"]\]"`. Skip xpaths that don't match this pattern (page, group xpaths).
         - Look up model in known_models["models"] (Tier 1) and registry (Tier 2).
         - If model found (Tier 1 or 2): check if referenced field exists in known fields. If not, emit E17 error.
         - If model not found in either: emit W6 warning: "Base model '{model}' not in known models or registry. Cannot validate field references."
       - Return (errors, warnings) tuple.
