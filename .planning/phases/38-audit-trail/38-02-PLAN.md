    def write(self, vals):

        if self.env.context.get('_audit_skip'):
            {# Early return path -- skip audit but still run cache/constraints/super #}

            result = super().write(vals)

{% endfor %}

        # Log AFTER super() succeeds

{% endif %}

**Add helper methods after write() (inside the model class, only for audited models):**

1. `_audit_tracked_fields(self)` -- `@api.model` method returning a set literal of auditable field names (rendered from `audit_field_names` context variable). Template: `return {{ audit_field_names | sort | list }}` rendered as a Python set literal.

2. `_audit_read_old(self, vals)` -- Reads current values for fields being written. Uses `self.sudo()` for safe reads. Handles field types per CONTEXT.md serialization rules:
   - Many2one: `[value.id, value.display_name] if value else False`
   - Date/Datetime: `str(value) if value else False`
   - Selection: `str(value) if value else False`
   - Others: raw value

3. `_audit_log_changes(self, old_values, vals)` -- Creates audit.trail.log entries via `self.env['audit.trail.log'].sudo().with_context(_audit_skip=True)`. Compares old vs new, serializes new values same way as old, creates one entry per record with changes dict `{"field": {"old": v, "new": v}}`.
