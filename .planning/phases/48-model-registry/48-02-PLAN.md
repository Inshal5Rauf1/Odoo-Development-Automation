    Write tests FIRST in test_cli_registry.py using Click's CliRunner (RED). Then implement:

    1. **CLI registry command group** in cli.py:
       - `@main.group()` for `registry` with docstring "Manage the cross-module model registry."
       - ALL registry imports MUST be lazy (inside functions) to preserve test_cli_lazy_imports
       - Helper `_find_registry_path()` that returns `Path(".planning/model_registry.json")` (relative to cwd)
       - Subcommands:
         - `registry list [--json]` — load registry, display modules/models or JSON output
         - `registry show <model_name>` — load registry + known models, display model details (fields, module, inherits, mixins)
         - `registry remove <module_name>` — load registry, remove module, save
         - `registry rebuild [--scan-root PATH]` — scan for __manifest__.py files, parse each, rebuild registry from scratch
         - `registry validate` — load registry + known models, run validate_comodels for all registered specs, run detect_cycles, print results with severity symbols (checkmark/warning/error), exit 1 if errors
         - `registry import --from-manifest PATH` — parse __manifest__.py with ast.literal_eval, parse model .py files with AST to extract _name and field definitions, register module

    2. **Post-render hook** in existing `render_module_cmd`:
       - AFTER successful `render_module()` call (inside try block, after render succeeds)
       - Lazy import ModelRegistry
       - Load registry + known models
       - Run validate_comodels(spec) — print warnings with "WARNING:" prefix, print errors with "ERROR:" prefix
       - Run infer_depends(spec) — print any inferred depends not already in spec
       - Call register_module(module_name, spec) — overwrites previous entries
       - Save registry
       - Print "Registry updated: +N models (module_name)"
       - If render_module() raises exception, registry update is skipped entirely (no partial state)

    3. **CLI output format** per CONTEXT.md:
       ```
       checkmark Spec validated (Pydantic)
       checkmark Module generated: uni_fee (4 models, 12 views)
       warning WARNING: comodel 'uni.scholarship' not in registry
       checkmark Registry updated: +4 models (uni_fee)
       ```

    5. Run tests (GREEN). All tests must pass including existing test_cli_lazy_imports.
