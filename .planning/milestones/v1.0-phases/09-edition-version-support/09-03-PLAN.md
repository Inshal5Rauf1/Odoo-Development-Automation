  <action>
    **Add check-edition CLI command:**

    Add import at top: `from odoo_gen_utils.edition import check_enterprise_dependencies`

    Add new click command `check-edition`:
    ```python
    @main.command("check-edition")
    @click.argument("spec_file", type=click.Path(exists=True))
    @click.option("--json", "json_output", is_flag=True, help="Output as JSON")
    def check_edition(spec_file: str, json_output: bool) -> None:
        """Check a module spec for Enterprise-only dependencies.

        Reads the depends list from a spec JSON file and reports any
        Enterprise-only modules with Community alternatives.
        """
    ```

    Implementation:
    1. Load spec JSON from spec_file
    2. Extract `depends` list from spec (default to `["base"]`)
    3. Call `check_enterprise_dependencies(depends)`
    4. If no warnings: print "All dependencies are Community-compatible." and exit 0
    5. If warnings: print each warning with module name, display name, and alternative (if any)
    6. With `--json`: output the warnings list as JSON
    7. Exit 0 always (warnings are informational per Decision B -- never blocks)

    **Update render command for version awareness:**

    In the `render` command function, after `env = create_renderer(template_dir)`, add version detection: if `context.get("odoo_version")` exists, use `create_versioned_renderer(context["odoo_version"])` instead. This allows `--var odoo_version=18.0` to trigger versioned template loading.

    Add import: `from odoo_gen_utils.renderer import create_versioned_renderer`

    **Update render-module command:**

    The `render_module()` function already reads odoo_version from spec internally (done in Plan 02). No change needed to `render_module_cmd` -- it passes spec to `render_module()` which handles version selection. But update the `template_dir = get_template_dir()` call to just pass the base templates dir (which is what `get_template_dir()` already returns).

    **Update list-templates command:**

    Update to list templates from all directories (shared/, 17.0/, 18.0/) or accept a `--version` option. Simplest: list shared/ templates plus version-specific dirs.

    **Do NOT change existing command signatures** -- backward compatible.
